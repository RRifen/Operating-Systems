Windows Shell

Windows Shell — это графический пользовательский интерфейс для ОС Microsoft Windows. Его элементы легко узнаваемы. Вот некоторые из них: рабочий стол, панель задач, меню «Пуск».

Windows Shell, который мы видим на наших компьютерах, представляет собой эволюцию интерфейса, которая началась с ОС Windows 95, выпущенной соответственно в 1995 году. Она неразрывно связана с появлением программы File Explorer (Проводник), компонентом Windows, который предоставляет графический интерфейс для доступа ко всем папкам и файлам.

macOS

Aqua — это графический интерфейс пользователя, главная тема оформления операционной системы macOS.

Стиль Аква и сам GUI были впервые представлены на Macworld Conference & Expo в январе 2000 года в Сан-Франциско. Новые компьютеры iMac и iBook были внешне обновлены, и поэтому был обновлен и сам интерфейс системы. Впервые этот стиль оформления был использован в iMovie 2 июля 2000 года, в котором использовались кнопки и полосы прокрутки в стиле Aqua.

Linux (GNOME)

GNOME — свободная среда рабочего стола для UNIX-подобных операционных систем. Разработчики GNOME ориентируются на создание полностью свободной среды, доступной всем пользователям вне зависимости от их уровня технических навыков, физических ограничений и языка, на котором они говорят.

Базовые приложения:

  • GNOME Terminal поддерживает все типичные функции эмулятора терминала, а также цветной вывод и события от мыши.
  • gedit — текстовый редактор с поддержкой Юникода. Поддерживает использование вкладок для представления нескольких документов в одном окне, подсветку синтаксиса для ряда компьютерных языков, и другие возможности.
  • Приложение Yelp предназначено для просмотра разного рода документации, установленной в системе. Yelp позволяет просматривать как справку по приложениям GNOME, так и стандартные справочные материалы man и texinfo.
